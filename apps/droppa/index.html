How VRF Works
1. Break is created on-chain (BreakFactory.sol)
2. All slots sold → break locked
3. VRF seed = keccak256(blockHash + breakId + timestamp)
4. Slots assigned using Fisher-Yates shuffle with VRF seed
5. Result verifiable by anyone on-chain
No break host can manipulate results after slots are sold. The VRF seed is derived from block data that no one can predict.
