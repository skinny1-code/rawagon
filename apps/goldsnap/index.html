sBTC Mint / Burn Flow
Mint: Send BTC to custodian → wait N confirmations → PSBT proof generated → call requestMint() → ops verify → mint() issued

Burn: Call requestBurn(amount, btcAddress) → ops build PSBT → 3-of-5 sign + broadcast → PoR updated

Security: 4-eyes on all withdrawals · Daily PoR reconciliation · Freeze on mismatch

Detection → GTX Flow
1. AS7341 scans spectral signature of sample
2. ADS1115 measures electrical conductivity
3. HX711 weighs sample (troy oz)
4. AI model classifies: gold confirmed / rejected
5. ATECC608A signs detection report (ECDSA)
6. Report pushed via WiFi → GoldSnap API
7. GoldMint.sol mints GTX = verified weight × purity
